## Introduction
SnapDrift is a lightweight, desktop-based YouTube video downloader built with Electron. It allows users to download multiple videos simultaneously, supporting multiple quality formats and MP3 conversion directly from your desktop.

## Features
* **Multi-Format Downloads:** Support for 1080p, 720p, 360p, and MP3 audio.
* **Batch Processing:** Download multiple videos at a time.
* **Custom Save Location:** Pick exactly where your files go.
* **Real-time Progress:** Track download percentages and conversion status.
* **Clean UI:** Built with BeerCSS for a modern, distraction-free experience.

## Installation

### For Users (Download .exe)
1. Go to the [Releases](https://github.com/AquariusSPoolsDev/snapdrift-yt-download/releases) page.
2. Download the latest exe file.
3. Run the file to start downloading.

## How To Use
1. **Paste URL:** Copy a YouTube link and paste it into the "Video URL" field.
2. **Fetch Metadata:** Click **Fetch Video** to load the title and thumbnail.
3. **Choose Format:** Select your preferred quality (e.g., 1080p or MP3).
4. **Select Location:** (Optional) Click the folder icon to change the download path.
5. **Download:** Click **Start Download**. The progress bar will update in the list below.

## Troubleshoot
* **Download Fails Immediately:** Ensure your internet connection is active and the video is not "Private" or "Age Restricted." Please try again later after some time.
